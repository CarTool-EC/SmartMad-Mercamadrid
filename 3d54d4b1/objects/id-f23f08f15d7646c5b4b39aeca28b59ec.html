Investigar como lo soluciona FIWARE: Smart Data Models: FIWARE promueve el uso de modelos de datos inteligentes que son reutilizables y compartidos. Estos modelos de datos pueden considerarse como una especie de "vocabulary hub" ya que proporcionan definiciones estándar y comunes para describir entidades y sus atributos en diferentes dominios. FIWARE Data Models: Estos modelos de datos son colecciones de esquemas JSON utilizados para describir diferentes entidades en varios dominios (como agricultura, transporte, etc.). Aunque no es un "hub" en el sentido de una plataforma centralizada, estos modelos de datos estándar facilitan la interoperabilidad y el intercambio de información.
