Azure Container Registry ()
Definition: Azure Container Registry handles private Docker container images as well as related content formats, such as Helm charts, OCI artifacts, and images built to the OCI image format specification. The solution suggests using a managed kubernetes service for deploying most of the runtimes, and this technology requries storing images in a container registry.
