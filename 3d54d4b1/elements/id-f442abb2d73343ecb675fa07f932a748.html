ADMS ()
Description: The Asset Description Metadata Schema (ADMS) is a standard metadata specification developed by the European Commission's ISA² program (Interoperability Solutions for European Public Administrations). ADMS is designed to facilitate the description and discovery of digital assets, particularly in the context of public administration.
