VMWare VSphere ()
Definition: VMware vSphere is VMware's virtualization platform, which transforms data centers into aggregated computing infrastructures that include CPU, storage, and networking resources. vSphere manages these infrastructures as a unified operating environment, and provides you with the tools to administer the data centers that participate in that environment. Source: VMWare Source reference: https://docs.vmware.com/en/VMware-vSphere/index.html Synonym: Example: The following implementation is an example on how this specific Architecture Building Block (ABB) can be instantiated as a Solution Building Block (SBB): VMWare VSphere is used as part of on premises data centres using Virtual Machines (Hypervisors) instead of using a full fledge cloud environment. eira:iopDimension: Structural IoP LOST view: TVI-Containers Infrastructure Enablers [Traceability] eira:iopSaliency: Containers Manager ABB is salient for technical interoperability due to its setting up the software tool to manage virtual public services and tools.
