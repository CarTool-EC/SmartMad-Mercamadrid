Azure Virtual Network ()
Definition: Azure Virtual Network is a service that provides the fundamental building block for your private network in Azure. An instance of the service (a virtual network) enables many types of Azure resources to securely communicate with each other, the internet, and on-premises networks. Source: Microsoft Example: The following implementation is an example on how this specific Architecture Building Block (ABB) can be instantiated as a Solution Building Block (SBB): A virtual network is similar to a traditional network that you'd operate in your own datacenter. But it brings extra benefits of the Azure infrastructure, such as scale, availability, and isolation. eira:iopDimension: Structural IoP LOST view: TVI-Networking Infrastructure Enablers [Traceability] eira:iopSaliency: Virtual Private Network ABB is salient for technical interoperability because it implements standard methods to build a secure network to exchange information in public administration.
